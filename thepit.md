## 一、核心玩法闭环

> **跳入天坑 → 战斗/任务/挖矿赚取熟练度 → 消耗熟练度锻造武器/学习武学 → 配置战备再战 → 境界提升解锁更高级内容**

## 二、经济系统：熟练度

### 2.1 熟练度产出途径

| 途径 | 基础数值 | 详细规则 |
| :--- | :--- | :--- |
| **在线时长** | 基础 3,530/小时 | **软上限**：第1小时100%，第2小时80%，第3小时60%，第4小时40%，第5小时及以后20%（最低不低于700/小时） |
| **击杀玩家** | 境界等级 × 500 + 100 | 被击杀者不掉落任何物品/熟练度，仅重生重置 |
| **任务系统** | 详见[第四章](#四任务系统) | 一次性引导任务 + 日常/周常 |
| **月卡加成** | 提升至 8,530/小时 | 软上限规则仍生效，仅提升基础值 |

### 2.2 熟练度消耗途径

| 消耗项 | 扣费方式 | 备注 |
| :--- | :--- | :--- |
| **境界升级** | 一次性扣除 | 10境数值递增（已单独策划） |
| **锻造武器** | **开始锻造时一次性全额扣除** | 中途取消按比例退还（见[3.4](#34-取消锻造规则)） |
| **学习武学** | 一次性扣除 | 通用武学便宜，门派专属昂贵 |
| **修炼身法** | 一次性扣除 | 被动/主动身法 |
| **黑市购买** | 支付卖家定价 | 系统收取5%~10%交易税（防通胀） |

### 2.3 防通胀机制

- 所有消耗均为**永久性消耗**（熟练度不会自动再生）
- 黑市交易税作为**系统回收阀**
- 月卡仅加速产出，不产出稀有材料

## 三、锻造系统

### 3.1 品质体系

| 品质等级 | 所需材料 | 词条数量 | 锻造耗时 | 定位 |
| :--- | :--- | :--- | :--- | :--- |
| **铁质** | 铁锭 × 5 | 0 ~ 1条 | 10分钟 | 新手入门，易获取 |
| **钻石质** | 钻石 × 3 | 1 ~ 2条 | 1.5小时 | 中期主力，性价比高 |
| **下界合金质** | 下界合金锭 × 2 | 2 ~ 3条 | 5小时 | 顶级神兵，长线追求 |

> **下界合金锭合成配方**：4个下界合金碎片 + 4个金锭 → 1个下界合金锭（与原版MC一致）

### 3.2 交互流程

1. 玩家与**武器商NPC**交互 → 打开锻造界面（箱子）
2. 界面展示：
   - **左侧**：武器类型列表（剑/刀/暗器/棍/拳套等）
   - **右侧**：选中类型后显示 → 所需材料 / 熟练度费用 / 锻造耗时 / 预计品质
3. 点击 **"开始锻造"** → 系统校验材料+熟练度 → 扣除 → 写入数据库 → 右侧记分板显示倒计时

### 3.3 后台锻造机制

- 锻造开始后，玩家**可自由战斗/挖矿/做任务**，锻造在后台进行
- 数据库记录字段：`UUID`、`武器ID`、`开始时间戳`、`总耗时（秒）`、`完成状态`
- 玩家下线/服务器重启不影响计时
- 基础同时锻造数量：**1把**（可付费解锁最多3把同时进行）

### 3.4 取消锻造规则

- 玩家可随时取消锻造任务
- 退还公式：
  - 退还熟练度 = 已消耗熟练度 × (已耗时 / 总耗时)
  - 退还材料 = 已消耗材料 × (已耗时 / 总耗时)（向下取整）
- **示例**：锻造5小时的合金剑，2.5小时取消 → 退还50%熟练度 + 50%材料

## 四、任务系统

### 4.1 一次性任务

每个玩家**仅可完成一次**，旨在引导熟悉所有系统并给予启动资源。

| 任务名称 | 要求 | 奖励熟练度 | 解锁条件 |
| :--- | :--- | :--- | :--- |
| 初次跳坑 | 进入天坑并完成1次击杀 | 5,000 | 无 |
| 初次挖矿 | 在矿石事件中挖掉1块铁矿石 | 8,000 | 完成"初次跳坑" |
| 初次锻造 | 锻造出第一把武器（任意品质） | 15,000 | 完成"初次挖矿" |
| 境界突破 | 升到锻皮境（第1境） | 30,000 | 完成"初次锻造" |
| 加入门派 | 选择第一个三流门派 | 20,000 | 达到锻皮境 |
| 门派晋升 | 从三流升到二流门派 | 50,000 | 达到化劲境（第6境） |
| 顶尖高手 | 从二流升到一流门派 | 100,000 | 达到藏锋境（第8境） |
| 武极之路 | 升到武极境（满境） | 200,000 | 达到破招境（第9境） |

### 4.2 日常/周常任务

每天刷新3个日常，每周刷新2个周常。

| 任务类型 | 示例 | 奖励熟练度 | 刷新周期 |
| :--- | :--- | :--- | :--- |
| 日常-击杀 | 击杀5名玩家 | 8,000 | 每日 |
| 日常-挖矿 | 挖掉10块矿石（不限种类） | 6,000 | 每日 |
| 日常-战斗时长 | 累计造成伤害10分钟 | 5,000 | 每日 |
| 周常-连杀 | 达成一次5连杀 | 40,000 | 每周 |
| 周常-锻造大师 | 锻造3把武器（至少精良品质） | 60,000 | 每周 |
| 周常-门派荣耀 | 使用门派专属武学击杀20人 | 50,000 | 每周 |

### 4.3 任务交互

- 玩家与**任务NPC**交互 → 查看可接任务列表（箱子UI）
- 接取后右侧记分板显示任务进度（如"击杀：3/5"）
- 完成后自动结算奖励，无需手动交任务

## 五、境界与门派系统

### 5.1 十境体系

| 阶段 | 境界名称（共10境） | 门派权限 |
| :--- | :--- | :--- |
| 初期 (1-5境) | 锻皮 → 淬血 → 凝劲 → 通脉 → 固元 | 只能选**三流门派（10个）** |
| 中期 (6-7境) | 化劲 → 洗髓 | 可加入**二流门派（4个）** |
| 后期 (8-10境) | 藏锋 → 破招 → 武极 | 可加入**一流门派（2个）** |

### 5.2 境界升级规则

- 每个境界升级**仅消耗熟练度**（具体数值已单独策划）
- 从三流到二流（第6境）：**不掉境界**
- 从二流到一流（第8境）：**不掉境界**
- **主动切换同级别门派**（如二流A切到二流B）：**掉1个小境界**（如化劲→凝劲，但不会掉到三流以下）

### 5.3 门派武学保留规则

- **通用武学**：全门派通用，切换后保留
- **门派专属武学**：切换门派后**禁用但保留数据**，切回原门派后自动恢复可用状态

## 六、矿石事件系统

### 6.1 生成逻辑

1. 每隔 **5~8分钟**（随机），系统扫描整个天坑地图的**所有露天方块**（顶部无遮挡）
2. 扫描范围：仅限**原石/石头/深板岩**等可替换方块
3. 每个符合条件的方块有 **1%~2%** 概率被选中
4. 被选中的方块记录：`坐标` + `原始方块ID`
5. 替换为矿石（概率分布）：
   - 铁矿（60%）→ 挖掉掉落**铁锭**
   - 金矿（25%）→ 挖掉掉落**金锭**
   - 钻石矿（12%）→ 挖掉掉落**钻石**
   - 远古残骸（3%）→ 挖掉掉落**下界合金碎片**

### 6.2 挖矿规则

- 玩家**只能破坏矿石**，其他方块无法破坏
- 挖掉后**直接掉落成品**（铁锭/金锭/钻石/下界合金碎片）
- 掉落瞬间，**原始方块自动补回**（不破坏地图结构）
- 矿石为**公共资源**，先挖先得（鼓励PVP争夺）

### 6.3 下界合金合成链路

- **4个下界合金碎片 + 4个金锭 → 1个下界合金锭**（与原版MC一致）
- 下界合金锭用于锻造**下界合金质**武器

## 七、战备仓库系统

### 7.1 仓库功能

- 每个玩家拥有独立仓库（存储武器/材料/丹药）
- 仓库UI：箱子界面，分页显示（武器页/材料页/丹药页）

### 7.2 战备配置

- 玩家在出生点（坑顶）打开**战备界面**
- 从仓库中选择：
  - 主武器 ×1
  - 副武器 ×1（可选）
  - 防具 ×1套（后续策划）
  - 丹药 ×N（快捷栏）
- 配置完成后点击"准备就绪" → 跳入天坑开始战斗
- **跳入后无法更换装备**，直到死亡重生或使用特定道具返回坑顶

### 7.3 装备丢失规则

- **玩家自身原因**（如主动丢弃、误操作）→ 自己承担，不补偿
- **服务器原因**（崩溃、回档、BUG）→ 后台日志核实后，由管理员补发同款装备

## 八、黑市交易系统

### 8.1 挂单规则

- 玩家在**黑市NPC**处上架武器/丹药
- 设定价格（熟练度）+ 数量
- 系统收取 **5%~10% 交易税**（从卖家收益中扣）
- 挂单有效期：7天，过期自动下架退还给卖家

### 8.2 购买规则

- 买家浏览黑市列表（箱子UI分页展示所有在售物品）
- 点击购买 → 系统校验买家熟练度 → 扣除（含税） → 物品转入买家仓库
- 卖家获得熟练度（扣除税费后自动到账）

## 九、内息与武学系统

### 9.1 内息机制

- 基础内息上限：100点
- **每秒自动恢复1%**（即100秒从0回满）
- 击杀玩家额外恢复：击杀者内息 +20%
- 使用技能消耗内息（具体数值因武学而异）

### 9.2 武学分类

| 类型 | 获取方式 | 消耗熟练度 | 冷却 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| 通用武学 | 所有门派可学 | 低 | 短 | "基础剑法"：单体伤害，消耗10内息，CD 5秒 |
| 门派专属 | 仅限特定门派 | 高 | 中 | 唐门"暴雨梨花针"：范围伤害，消耗30内息，CD 20秒 |
| 身法（被动） | 所有门派可学 | 中 | 无 | "梯云纵"：跳跃高度+30%，永久生效 |
| 身法（主动） | 所有门派可学 | 中 | 长 | "凌波微步"：移速+50%，持续10秒，CD 60秒，消耗20内息 |

## 十一、付费点设计

| 付费项 | 效果 | 原则 |
| :--- | :--- | :--- |
| **月卡** | 在线熟练度从 3,530/h → 8,530/h | 加速但不氪金则无敌 |
| **锻造槽位** | 解锁第2/3个同时锻造队列 | 节省时间，不直接提升战力 |
| **外观/特效** | 武器光效、击杀播报、门派披风 | 纯装饰，不影响数值 |

> **核心原则**：付费无法直接购买武器/武学/境界，只能加速时间成本。

## 十二、技术实现关键点

### 12.1 锻造数据持久化

- 存储字段：`uuid`, `weapon_id`, `start_timestamp`, `total_seconds`, `status`
- 玩家打开锻造UI时实时查询，已完成则显示"取走武器"，进行中则显示剩余时间

### 12.2 矿石生成性能优化

- 建议采用**异步扫描** + **分帧处理**（每tick扫描100个方块）
- 或采用**区域预生成**：提前标记可替换方块坐标池，事件触发时直接从池中随机抽取

### 12.3 Geyser基岩版适配

- 箱子UI天然兼容，无需额外开发
- 记分板/ActionBar在基岩版同样支持
